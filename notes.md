# Hamzah LLC Website - Final Design

## Brand Colors

| Mode | Primary | Accent | Background |
|------|---------|--------|------------|
| **Light** | `#4A1F6E` (Dark Purple) | `#FFB951` (Gold) | `#fdfcfe` (Very subtle purple tint) |
| **Dark** | `#FFB951` (Gold) | `#A76FD9` (Light Purple) | `#0d0710` (Very dark purple) |

### Color Palette
- `#0d0710` - Darkest background
- `#1a0d20` - Dark secondary background  
- `#2a1235` - Dark tertiary
- `#3a1847` - Dark elevated
- `#4A1F6E` - **Primary brand color**
- `#A76FD9` - **Secondary brand color**
- `#FFB951` - **Accent (gold)**

## Key CSS Classes

```css
/* Brand colors in both modes */
Light mode accent: #4A1F6E (purple)
Dark mode accent: #FFB951 (gold)

/* Gradient text */
.text-gradient - Purple to Gold gradient

/* Badge */
.badge - Uses brand colors

/* Icon box */
.icon-box - Purple in light, Gold in dark

/* Footer background */
#0d0710 (darkest purple)
```
